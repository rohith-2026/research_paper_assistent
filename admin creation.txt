How to Create a New Admin User

Option A: Seed Script (recommended for first admin)
1) Set environment variables:
   - ADMIN_SEED_EMAIL
   - ADMIN_SEED_PASSWORD (min 8 characters)
2) Run:
   python -m app.db.run_seed_data

PowerShell example:
$env:ADMIN_SEED_EMAIL="admin@example1.com"
$env:ADMIN_SEED_PASSWORD="ChangeMe123!"
python -m app.db.run_seed_data

Notes:
- Script creates the admin only if it does not already exist.
- Also inserts a default admin_settings document if missing.
